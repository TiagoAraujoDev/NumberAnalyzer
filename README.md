Besides, it's good to have an editor to work with the code like [VSCode](https://code.visualstudio.com/)

<h3 id="objetivo">🎯 Goals</h3>

- Getting better at programming in the only way it works...practicing
- In this project I was concerned with the validation of the data provided by the user. It had to be a number between 0 and 100, it should only be added once and if the input field is blank it should alert the user. Beside that the CSS was very expermental by styling scrollbar and the input.
